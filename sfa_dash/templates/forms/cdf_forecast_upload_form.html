CSV JSON

Forecast data in CSV format should follow the formatting of the example below.

# optional header, ignored by Solar Forecast Arbiter 
timestamp,value
2018-11-22T12:00:00Z,10.23
2018-11-22T12:05:00Z,10.67

Forecast data in JSON format should follow the formatting of the example below.

{ 
    “id”: “testid”, # optional, for uploader’s record 
    “values”: [
        { 
           “timestamp”: “2018-11-22T12:00:00Z”,
            “value”: 10.23, 
         }, 
        { 
           “timestamp”: “2018-11-22T12:05:00Z”,
            “value”: 10.67, 
         }            
    ] 
}
